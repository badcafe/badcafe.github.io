  • Create a nested reviver that can convert a value and its nested values.

    Usage

    const personReviver = Jsonizer.reviver<typeof person>({
        birthDate: Date
    });

    Type Parameters

    • Target
    • Source = Target
    • Any extends string = "*"
    • Self extends string = "."
    • Delim extends string = Source extends any[] ? "-" : "/"

    Parameters

    • mappers: Mappers<Target, Source, Any, Self, Delim>

      Mapper for the current value and children values

    Returns Reviver<Target>

    A reviver that can be passed to JSON.parse() or that can be used as a reviver object on which one can directly call .revive(). That reviver can also be itself serialized and revived.

    Param Type

    Target - The actual class to revive

    Param Type

    Source - The JSON representation of Target is by default the structural Target itself
