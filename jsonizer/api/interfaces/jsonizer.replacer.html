Interface Replacer<Type>

The replacer to use as a second argument in JSON.stringify().

After stringification, a reviver is available to restore the original data structure. This function is a stateful function that can be used once and only once in JSON.stringify().

paramtype

Type The type to replace, to have it in the reviver.
